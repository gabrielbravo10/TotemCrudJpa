# TotemCrudJpa


TotemCrudJpa is an example project of a crud build in Java.

## 1. Technical specifications

- Java
- Jsp
- MySql Connector
- JSTL
- MVC
- Servlet
- Hibernate
- JPA

## 2. Build

- Create Database
- Execute script `src/main/resources/META-INF/setup.sql`
- Edit the file `src/main/resources/META-INF/persistence.xml` with data related to your environment
